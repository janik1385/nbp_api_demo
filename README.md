# NBP Web API Demo

Przykładowa implementacja klasy pobierającej aktualny kurs waluty euro przy pomocy NBP Web API.<br />
Przykład działa z wersją php 7.2 lub wyższą.<br />
<br />
Instalacja:<br />
<code>
git clone https://github.com/janik1385/nbp_api_demo.git
</code><br />
<code>
cd nbp_api_demo
</code><br />
<code>
composer install
</code>
<br />
